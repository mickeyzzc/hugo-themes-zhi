# Creative Commons License

Display a Creative Commons license badge.

## Configuration

```toml
[params.creativeCommons]
  enable = false
  license = "by-nc-sa"
  size = "normal"
```

## Usage

Set `enable` to true. Choose a license from: by, by-sa, by-nc, by-nc-sa, by-nd, by-nc-nd.
Set `size` to either "normal" or "big".
