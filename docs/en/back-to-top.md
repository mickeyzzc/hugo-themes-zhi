# Back to Top Button

Floating back-to-top button for easy navigation.

## Configuration

```toml
[params.features]
  backToTop = true

[params.backToTop]
  scrollPercent = true
```

## Usage

The button automatically appears after scrolling down the page. When `scrollPercent` is enabled, it shows the scroll percentage.
