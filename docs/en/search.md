# Search

Local search via XML index and search UI. Includes keyboard shortcut (/) to focus search.

## Overview

This feature provides a local search capability that indexes your site content and allows users to search through posts, pages, and taxonomies. It uses an XML index generated by Hugo and a client-side search UI for instant results.

## Configuration

```toml
[params.features]
  search = true  # Default: true

[outputs]
  home = ["HTML", "RSS", "SearchIndexes"]

[outputFormats.SearchIndexes]
  mediatype     = "application/xml"
  baseName      = "searchindexes"
  isPlainText   = false
```

## Usage

This feature requires no special markup in content. It automatically:
- Generates a searchable XML index at `/index.xml`
- Provides a search UI accessible via the search icon in header or `/` shortcut
- Supports fuzzy matching and highlights results
- Works with Hugo's built-in taxonomies (categories, tags)
