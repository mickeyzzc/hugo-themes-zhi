# 问候语

基于时间的问候语功能，根据访问时间显示不同的问候（早上好/中午好/下午好/晚上好），仅在侧边栏启用时显示。

## 配置

```toml
[params.features]
  sidebar = true          # 侧边栏必须启用才能显示问候语
```

## 用法

当侧边栏启用时，问候语会自动显示在侧边栏头部。问候语根据访问者的本地时间动态切换：
- 凌晨 0:00 - 11:59：早上好
- 中午 12:00 - 13:59：中午好
- 下午 14:00 - 17:59：下午好
- 晚上 18:00 - 23:59：晚上好

该功能使用 i18n 键实现，可在 `i18n/zh-cn.toml` 中自定义文本。
